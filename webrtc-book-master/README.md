#web rtc 理论
##web rtc 认识
web rtc 是一个由google发起的实时通信解决方案、其中包含了音视频采集、 编解码、数据传输、音视频展示等功能。特点跨平台 免费 实时传输 打洞能力
信令主要用来转发会话双方的媒体信息和网络信息
##web rtc 通话原理
信令服务器转发
媒体协商
sdp描述多媒体连接内容的协议
NAT:为了解决IPv4下的IP地址匮乏而出现的一种技术
ICE 框架
STUN
